# 🔐 PassGen Pro - Ultimate Password Generator

An incredible, modern, and sleek password generator and analyzer mobile app built with React Native and Expo SDK 54.0.0.

## ✨ Features

- **🎨 Beautiful Modern UI**: Sleek, viral-worthy design with smooth animations
- **🌓 Dark Mode**: Full dark mode support with seamless theme switching
- **🔒 Smart Password Generation**: Customizable password generation with multiple character types
- **📊 Real-time Strength Analysis**: Advanced password strength analyzer with visual feedback
- **📱 Haptic Feedback**: Enhanced user experience with tactile feedback
- **📋 One-Tap Copy**: Quickly copy passwords to clipboard
- **⚡ Smooth Animations**: Beautiful transitions and interactive elements using Reanimated
- **💡 Security Tips**: Built-in best practices for password security
- **🎯 Customizable Length**: Choose from 8 to 32 character passwords
- **🎭 Character Options**: Toggle uppercase, lowercase, numbers, and symbols

## 🚀 Getting Started

### Prerequisites

- Node.js (v20.17.0 or higher recommended)
- npm or yarn
- Expo Go app on your mobile device (for testing)

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Run on your device:
   - **iOS**: Press `i` in the terminal or scan QR code with Camera app
   - **Android**: Press `a` in the terminal or scan QR code with Expo Go app
   - **Web**: Press `w` in the terminal

## 📱 Commands

- `npm start` - Start the Expo development server
- `npm run android` - Run on Android device/emulator
- `npm run ios` - Run on iOS device/simulator (macOS only)
- `npm run web` - Run in web browser

## 🎯 How It Works

### Password Generation
The app generates cryptographically random passwords using customizable character sets:
- Uppercase letters (A-Z)
- Lowercase letters (a-z)
- Numbers (0-9)
- Special symbols (!@#$%^&*()_+-=[]{}|;:,.<>?)

### Strength Analysis
The password analyzer evaluates passwords based on:
- Length (longer is better)
- Character diversity (mixing different types)
- Unique character ratio
- Complexity patterns

Strength ratings:
- ⚠️ **Very Weak** - Needs significant improvement
- 🔶 **Weak** - Below recommended standards
- 🟡 **Fair** - Acceptable but could be better
- ✅ **Good** - Meets security standards
- 🛡️ **Strong** - Excellent security
- 💪 **Very Strong** - Maximum security
