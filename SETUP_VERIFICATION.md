## 🚀 How to Run

### Quick Start
```bash
npm start
```

Then:
- Press `i` for iOS simulator
- Press `a` for Android emulator  
- Press `w` for web browser
- Scan QR code with Expo Go app on your device

### Platform-Specific
```bash
npm run android  # Android
npm run ios      # iOS (macOS only)
npm run web      # Web browser
```
